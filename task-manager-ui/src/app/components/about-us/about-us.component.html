This is a POC on Task management. The idea is to build a simple task management application that helps users organize their tasks and get things done. The application will allow users to create tasks, set due dates, and track their progress. The application will be designed to help users stay organized and focused on what matters most. The application will be built using Angular and CDS Web Components. The application will have the following features:
  • Users can create tasks
  • Users can set due dates for tasks
  • Users can track their progress on tasks
  • Users can mark tasks as completed
  • Users can delete tasks
